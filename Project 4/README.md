# 📊 Project 4: E-Commerce Sales Dashboard (Tableau)

## 📌 Overview

This project presents an interactive E-Commerce Sales Dashboard developed in Tableau to analyze sales performance and provide actionable business insights. The dashboard combines key performance indicators (KPIs) and interactive visualizations to help users understand revenue trends, customer behavior, product performance, payment methods, and order status distribution.

---

## 🎯 Objectives

The objectives of this project were to:

- Analyze overall sales performance.
- Monitor monthly revenue trends.
- Identify top-performing products.
- Examine customer purchasing behavior.
- Compare payment method usage.
- Visualize order status distribution.
- Build an interactive dashboard for business decision-making.

---

## 🛠️ Tools Used

- Tableau Desktop
- Microsoft Excel

---

## 📂 Dataset

The dashboard was created using a cleaned e-commerce sales dataset containing information such as:

- Order ID
- Customer ID
- Product
- Quantity
- Total Price
- Order Date
- Payment Method
- Order Status
- Referral Source

---

## 📈 Dashboard Features

### Key Performance Indicators (KPIs)

- Total Revenue
- Total Quantity Sold
- Average Order Value
- Total Orders
- Unique Customers

### Visualizations

- Revenue by Month (Line Chart)
- Revenue by Product (Horizontal Bar Chart)
- Revenue by Payment Method (Bar Chart)
- Order Status Distribution (Donut Chart)
- Total Price Distribution (Histogram)

### Interactive Filters

- Product Filter

---

## 🔍 Key Insights

The dashboard enables users to:

- Monitor monthly sales performance.
- Identify products generating the highest revenue.
- Compare revenue across different payment methods.
- Understand the distribution of order statuses.
- Track customer purchasing patterns.
- Gain a quick overview of business performance through KPI cards.

## 🧹 Data Preparation

Before visualization, the dataset was:
