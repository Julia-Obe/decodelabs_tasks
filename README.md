# DecodeLabs Data Analytics Internship Projects

## Overview

This repository contains three practical data analytics projects completed during my DecodeLabs Data Analytics Internship. Together, these projects demonstrate the end-to-end data analytics workflow—from data cleaning and exploratory data analysis (EDA) to extracting business insights using SQL.

Throughout these projects, I worked with real-world sales data to improve data quality, uncover meaningful patterns, and answer business questions using Python and PostgreSQL.

# Project 3 – SQL Sales Data Analysis

### Objective
Use SQL to answer business questions and generate insights from the cleaned sales dataset.

### SQL Concepts Applied
- SELECT
- FROM
- WHERE
- GROUP BY
- ORDER BY
- COUNT()
- SUM()
- AVG()
- DISTINCT
- LIMIT

### Business Questions Answered
- Total number of orders
- Number of unique customers
- Total revenue generated
- Average order value
- Highest and lowest selling products
- Most frequently used payment method
- Payment method generating the highest revenue
- Top referral source
- Revenue by product
- Revenue by payment method
- Product ranking
- Customer purchasing insights
- Order status analysis
- Coupon usage
- High-value transactions

### Key Findings
- Total Orders: **1,200**
- Unique Customers: **1,189**
- Total Revenue: **1,264,761.96**
- Average Order Value: **1,053.97**
- Highest Selling Product: **Chair**
- Lowest Selling Product: **Phone**
- Most Used Payment Method: **Online**
- Highest Revenue Payment Method: **Credit Card**
- Top Referral Source: **Instagram**
